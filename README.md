# Laboratorio Odontotecnico Roso Marcello - Sito Web

Repository: [https://github.com/contattirosomarcello-blip/laboratorio-odontotecnico](https://github.com/contattirosomarcello-blip/laboratorio-odontotecnico)

Sito web professionale per il Laboratorio Odontotecnico Roso Marcello a Torino. Showcase dei servizi, portfolio e contatti.

## Funzionamento
1. **Frontend**: HTML/CSS/JavaScript per presentare l'azienda in modo professionale.
2. **Backend**: Server Node.js/Express per gestire le richieste e i CORS.
3. **Hosting**: Distribuito su Render per la disponibilità 24/7.

## 🛠 Setup Rapido

1. **GitHub**: Carica i file escludendo i segreti tramite `.gitignore`.
2. **Render**: Crea un Web Service per il backend.
3. **npm**: Installa le dipendenze:
   ```bash
   npm install
   npm start
   ```

### Installazione Locale
Se desideri testare il bridge in locale:
```bash
npm install
npm start
```
