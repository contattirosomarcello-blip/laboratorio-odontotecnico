# Configurazione Ambiente

Per il corretto funzionamento su Render, impostare le seguenti variabili:

- `GAS_URL`: URL della Web App di Google Apps Script (terminante in /exec).
- `ADMIN_TOKEN`: Token di accesso per l'Admin Hub (`DentalTech_2024_Safe!#`).

### Proprietà Script (Google Apps Script)
- `BREVO_API_KEY`: Chiave API fornita da Brevo.
- `TELEGRAM_TOKEN`: Token del bot fornito da BotFather.
